# 辅助工具安装
## compass安装 
> http://compass-style.org/

	
```
$ gem update --system 
$ gem install compass 
```

> 问题：由于国内网络原因（你懂的），导致 rubygems.org 存放在 Amazon S3 上面的资源文件间歇性连接失败。解决方案RubyGems 镜像-淘宝网 http://ruby.taobao.org/

## gulp安装
	
	$ npm install --global gulp
